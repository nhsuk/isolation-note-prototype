You do not need an isolation note

You can only get an isolation note if:

  • you have been told to self-isolate by the NHS website, NHS Test and Trace, NHS 111, 111 online, NHS inform, NHS 111 Wales, Covid-19 NI app or Public Health Agency (Northern Ireland)

If you are not sure if you need to self-isolate, get the latest NHS advice on coronavirus.
